Development with Docker

Scripts are provided to build a Docker image and create a Docker container. The container can be used as a controlled development environment.

Build Image and Create Container

 
# Build image
$> npm run build-image

> toga@0.1.0 build-image /Users/jfavr1/dev/experimental/toga
> docker image build -t toga:dev .

Restarting and Stopping the Container

 
# Stop the container
$> npm run stop-container

# Restart the container
$> npm run restart container

Connecting to the Container

The containerized application is made available via localhost on port 8080. The local source files are mounted inside the container, so changes to them will immediately be reflected.

Changes to the package.json file, including the adding of dependencies necessitate the rebuilding of the Docker image.

First stop the container, then rebuild the image, finally run a container with the new base image.

# Stop the container (container is deleted)
$> npm run container-stop

# Rebuild the image
$> npm run build-image

# Start a new container
$> npm run start-container
